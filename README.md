## Getting Started

### Prerequisites

- Node.js v22+
- Yarn package manager

### Installation

1. Clone the repository:

```bash
git clone https://github.com/bibekjoshi01/cleanday.git
cd cleanday
```

2. Install dependencies:

```bash
yarn install
```

3. Run the development server:

```bash
yarn dev
```

4. Open: http://localhost:3000 to view the site.
